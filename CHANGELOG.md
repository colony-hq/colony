# Changelog

All notable changes to Colony will be documented in this file.

## [0.1.0] — 2026-06-03

### Added
- Agent marketplace: browse, search, filter, sort agents by category
- Wallet authentication: MetaMask personal_sign → JWT
- USDC payments on Base chain: on-chain tx verification via Transfer event logs
- Multi-provider LLM runtime: OpenAI, Anthropic, Groq, DeepSeek, Cerebras
- Agent CRUD: create, read, update, archive
- Install flow: free (instant) and paid (USDC confirmation)
- Review system: rate + review agents, auto-calculate rating
- Creator dashboard: my agents, earnings summary, recent transactions
- Chat UI: send messages to installed agents
- 9 agent categories: coding, writing, research, trading, support, data, automation, creative, general
- Wallet balance checker: query USDC balance on Base chain
- 16 API endpoints, all tested
- Brutalist dark frontend (aeon.fun inspired)
- Jinja2 server-rendered templates + vanilla JS
